# 🎮 Текстовая новелла на kotlin



---

## 📖 О проекте

Это учебный проект для изучения языка **Kotlin** на практике. Игра представляет собой текстовый квест в жанре "choose your own adventure", где игроку предстоит прожить один день из жизни студента в типичном российском колледже.

> ⚠️ **Внимание:** Cодержит ненормативную лексику, Неприемлемый юмор. 18+

---

## 🎯 Цель проекта

Изучение Kotlin через создание рабочего проекта, я смог пока сделать:
- ✅ Реальную практику синтаксиса
- ✅ Понимание работы функций и условий
- ✅ Основы ООП (в процессе добавления)
- ✅ Работу с вводом/выводом данных
- ✅ Рандомизацию и вероятности

## 🚀 Используемые возможности Kotlin

### Базовые темы:
| Тема | Описание |
|------|----------|
| `var/val` | Переменные и константы для состояния игры |
| `if/else` | Ветвление логики в зависимости от выбора игрока |
| `when` | Аналог switch-case для обработки вариантов выбора |
| `do-while` / `while` | Циклы для повторных попыток ввода |
| `fun` | Функции с возвращаемым значением и без |
| `readlnOrNull()` | Чтение пользовательского ввода |
| `println()` | Вывод текста в консоль |
| `Thread.sleep()` | Задержки для повествования |

### Продвинутые темы (в разработке):
| Тема | Описание |
|------|----------|
| `data class` | Хранение состояния игры и персонажей |
| `List` / `MutableList` | Списки предметов и доступных действий |
| `sealed class` | Типизированные события игры |
| Лямбда-выражения | Колбэки для событий |
| ООП | Классы для уровней, персонажей, локаций |

## 🎮 Геймплей

### Сюжет
Димка — типичный студент колледжа. Ему нужно:
1. Пережить утро 
2. Доехать до колледжа 
3. Выдержать первую пару 
4. Пообедать в кафе "Долма"
5. Дожить до вечера и не быть отчисленным

### Механики
- **Ветвящийся сюжет** — каждый выбор влияет на концовку
- **Система удачи** — рандом определяет успех действий
- **Состояния** — переменные влияют на развитие событий
- **Несколько концовок** — от "выжил" до "отчислен с позором"

### Пример выбора:
```
Настало время его первого сложного выбора за этот день:
Что стоит сделать димону в первую очередь?

ВВЕДИТЕ ЖЕЛАЕМОЕ ДЕЙСТВИЕ:
→ *******
→ почистить зубы
→ поесть
→ лечь обратно спать
```

## 🛠 Установка и запуск

### Требования
- **JDK 11+** (рекомендуется JDK 17)
- **IntelliJ IDEA** или любой другой редактор с поддержкой Kotlin
- **Kotlin** (встроен в IntelliJ IDEA)

### Шаги:
1. **Склонируйте репозиторий:**
   ```bash
   git clone https://github.com/ТВОЙ_НИК/PavlovGameKotlinIntelliJ.git
   cd PavlovGameKotlinIntelliJ
   ```

2. **Откройте проект в IntelliJ IDEA:**
   - File → Open → Выберите папку проекта

3. **Запустите игру:**
   - Найдите файл `Main.kt`
   - Нажмите `Run` (зелёная стрелка) или `Shift + F10`

4. **Играйте:**
   - Вводите команды в консоль
   - Читайте повествование
   - Принимайте решения
